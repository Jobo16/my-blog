# Blog Images Directory

这个目录用于存放博客文章中使用的图片和其他媒体文件。

## 目录结构
- `first-post/` - 第一篇文章的图片
- `frontend-best-practices/` - 前端最佳实践文章的图片
- `staying-motivated/` - 保持动力文章的图片

## 使用方法
1. 为每篇文章创建一个独立的文件夹
2. 将图片放入对应的文件夹中
3. 在 Markdown 文件中使用相对路径引用图片：
   ```markdown
   ![图片描述](/blog-images/文章文件夹名/图片名.jpg)
   ```
